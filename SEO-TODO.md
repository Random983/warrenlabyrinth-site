# SEO Follow-Up Note

When the site has more real articles, do this:

1. Open Google Search Console
2. Add the site property for `warrenlabyrinth.com`
3. Add the site property for `www.warrenlabyrinth.com`
4. Submit the sitemap: `https://www.warrenlabyrinth.com/sitemap.xml`
5. Use URL Inspection for the homepage and important articles
6. Click `Request Indexing`

Reason:

Google may not index a brand new site quickly. It is better to do this once there are several real blog posts published.
